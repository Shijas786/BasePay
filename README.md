## Features
- Gasless crypto transfers
- Simple WhatsApp commands (`/send`, `/balance`, `/tip`)
- P2P payment flows
- Easy onboarding (WhatsApp-first)

## Tech Stack
- Node.js + Express
- Twilio WhatsApp API
- Ethers.js + Base RPC
- Biconomy (for gasless)
- MongoDB / Firebase

## Setup
1. Clone this repo
2. Install dependencies: `npm install`
3. Setup Twilio Sandbox
4. Start: `npm run dev`
